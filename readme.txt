Distance formula calculator finds the distance between two points on a plane based on their coordinates. The calculator uses the Euclidean distance formula.  

[https://www.calculator.io/distance-formula-calculator/](https://www.calculator.io/distance-formula-calculator/)

== Usage ==

Add the Distance Formula Calculator shortcode to your page, post or sidebar:

`[ci_distance_formula_calculator]`

Add the following code to your template where you would like the Distance Formula Calculator to appear:

`<?php display_ci_distance_formula_calculator(); ?>`

== Installation ==

1. Upload the Distance Formula Calculator /ci_distance_formula_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Distance Formula Calculator plugin through the "Plugins" menu in WordPress.
