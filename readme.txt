== Description ==

Distance formula calculator finds the distance between two points on a plane based on their coordinates. The calculator uses the Euclidean distance formula.  

[https://www.calculator.io/distance-formula-calculator/](https://www.calculator.io/distance-formula-calculator/)

__Distance Formula Calculator Shortcode__

`[ci_distance_formula_calculator]`

== Installation ==

1. Upload the /ci_distance_formula_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Distance Formula Calculator](https://www.calculator.io/distance-formula-calculator/ "Distance Formula Calculator Homepage") plugin through the "Plugins" menu in WordPress.

== Distance Formula Calculator Usage ==

* Add the shortcode `[ci_distance_formula_calculator]` to your page, post or sidebar.
* Add the following code: `<?php display_ci_distance_formula_calculator(); ?>` to your template where you would like the Distance Formula Calculator to appear.
